dryxAstroTools.clean_supernova_name(dbConn, log, snName)¶

Given a SN name as a string, this function will attempt to clean up the name so that it is somewhat homogeneous with SN/transient from the same survey/atel system.

Key Arguments:
  • dbConn – mysql database connection
  • log – logger
  • snName – sn name to be cleaned (string)
Return:
  • snName – cleaned sn name (string)
dryxAstroTools.create_cbet_url(dbConn, log, cbet)[source]¶
Generates the CBET URL given a CBET number.
The URL contains a fiddly middle number before the CBET txt file - this function calculates this number before returning the valid URL.
Key Arguments:
  • dbConn – mysql database connection
  • log – logger
  • cbet – cbet number
Return:
  • cbetUrl – CBET URL
